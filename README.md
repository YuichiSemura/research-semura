# Research Repository - Yuichi Semura

博士論文に向けた研究テーマ探索・管理用リポジトリ

## 研究の方向性

**メインテーマ候補**: LLM生成コードにおけるクローン問題

### 背景

- 過去の研究: CCFinderSW (Clone Detection Tool with Flexible Multilingual Tokenization)
- 専門領域: ソースコードの静的解析、プログラミング言語

### 問題意識

- LLM/バイブコーディング時代におけるソースコードのあり方の変化
- LLM生成コードのクローン問題（学習データ由来の類似コード）
- 著作権・ライセンス問題
- 生成コードの品質保証・追跡可能性
